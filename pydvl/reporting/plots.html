spearman_correlation(vv: List[OrderedDict], num_values: int, pvalue: float)[source]#

Simple matrix plots with spearman correlation for each pair in vv.

Parameters:

  • vv – list of OrderedDicts with index: value. Spearman correlation is computed for the keys.

  • num_values – Use only these many values from the data (from the start of the OrderedDicts)

  • pvalue – correlation coefficients for which the p-value is below the threshold pvalue/len(vv) will be discarded.
