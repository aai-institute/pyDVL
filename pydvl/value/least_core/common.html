lc_solve_problem(problem: LeastCoreProblem, *, u: Utility, algorithm: str, non_negative_subsidy: bool = False, solver_options: Optional[dict] = None, **options) → ValuationResult[source]#

Solves a linear problem prepared by mclc_prepare_problem(). Useful for parallel execution of multiple experiments by running this as a remote task.

See exact_least_core() or montecarlo_least_core() for argument descriptions.

lc_solve_problems(problems: Sequence[LeastCoreProblem], u: Utility, algorithm: str, config: ParallelConfig = ParallelConfig(backend='joblib', address=None, n_cpus_local=None, logging_level=30, wait_timeout=1.0), n_jobs: int = 1, non_negative_subsidy: bool = True, solver_options: Optional[dict] = None, **options) → List[ValuationResult][source]#

Solves a list of linear problems in parallel.

Parameters

  • u – Utility.

  • problems – Least Core problems to solve, as returned by mclc_prepare_problem().

  • algorithm – Name of the valuation algorithm.

  • config – Object configuring parallel computation, with cluster address, number of cpus, etc.

  • n_jobs – Number of parallel jobs to run.

  • non_negative_subsidy – If True, the least core subsidy \(e\) is constrained to be non-negative.

  • solver_options – Additional options to pass to the solver.

Returns

List of solutions.
