knn

This module contains Shapley computations for K-Nearest Neighbours.

knn_shapley(data: Dataset, model: KNeighborsClassifier, *, progress: bool = True) → OrderedDict[str, float][source]

Computes exact Shapley values for a KNN classifier.

This implements the method described in Jia et al.1. It exploits the local structure of K-Nearest Neighbours to reduce the number of calls to the utility function to a constant number per index, thus reducing computation time to \(O(n)\).

Parameters

  • data – A pydvl.utils.dataset.Dataset object with a training / test split.

  • model – A KNN model to extract parameters from. The object will not be modified nor used other than to call get_params()

  • progress – Whether to display a progress bar.

Returns

An OrderedDict of data indices and their values.
