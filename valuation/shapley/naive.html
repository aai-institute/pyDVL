naive

permutation_exact_shapley(u: Utility, *, progress: bool = True) → OrderedDict[str, float][source]

Computes the exact Shapley value using permutations. When the length of the training set is > 10 it prints a warning since the computation becomes too expensive. Used mostly for internal testing and simple use cases. Please refer to the montecarlo methods for all other cases.

Parameters

  • u – Utility object with model, data, and scoring function

  • progress – set to True to use tqdm progress bars.

Returns

OrderedDict of exact Shapley values

combinatorial_exact_shapley(u: Utility, *, progress: bool = True) → OrderedDict[str, float][source]

Computes the exact Shapley value using the combinatorial definition. When the length of the training set is > 20 it prints a warning since the computation becomes too expensive. Used mostly for internal testing and simple use cases. Please refer to the montecarlo methods for all other cases.

Parameters

  • u – Utility object with model, data, and scoring function

  • progress – set to True to use tqdm progress bars.

Returns

OrderedDict of exact Shapley values
