class ValuationResult(algorithm: Callable, status: ValuationStatus, values: NDArray[np.float_], stderr: Optional[NDArray[np.float_]] = None, data_names: Optional[Sequence[str]] = None, sort: Optional[SortOrder] = None)[source]ÔÉÅ

Bases: Sequence

Objects of this class hold the results of valuation algorithms.

Results can be sorted with sort(). Note that sorting values affects how iterators and the object itself as Sequence behave: values[0] returns a ValueItem with the highest or lowest ranking point if this object is sorted by descending or ascending value, respectively. If unsorted, values[0] returns a ValueItem for index 0.

Similarly, iter(valuation_result) returns a generator of ValueItem in the order in which the object is sorted.

Parameters

  • algorithm ‚Äì The method used.

  • status ‚Äì The end status of the algorithm.

  • values ‚Äì An array of values, data indices correspond to positions in the array.

  • stderr ‚Äì An optional array of standard errors in the computation of each value.

  • data_names ‚Äì Names for the data points. Defaults to index numbers if not set.

  • sort ‚Äì Whether to sort the values. See above how this affects usage as an iterable or sequence.

Raises

ValueError ‚Äì If data names and values have mismatching lengths.
